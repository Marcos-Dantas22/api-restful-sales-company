# API RESTFUL

Este projeto é uma implementação de uma API RESTFUL

## Tecnologias utilizadas

- **Python**
- **FastAPI**
- **Docker**
- **PostgreSQL**
- **Pytest**
- **Sentry**
- **SQLAlchemy ORM**
- **Alembic migrations**
- **python-jose JWT (JSON WEB TOKEN)**
- **passlib cript passwords**
